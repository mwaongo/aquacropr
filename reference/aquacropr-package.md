# aquacropr: R interface to the FAO AquaCrop crop water productivity model (v7.0+)

R interface to AquaCrop v7.0 and higher, the FAO crop water productivity
model. Streamlines reading, writing, and managing AquaCrop input/output
files for batch simulations, climate scenario analysis, and crop yield
assessment.
